# MoreTooltips
Show item's info when mouse hover
## Feature
* Just need fabric API
* Burn Time Tooltip
* Show item's ID
* Durability Tooltip
* Mining Level of tool
* Hunger/Saturation Tooltip
* Max Stack Size Tooltip
* NBT Data Tooltip
* Registry Name Tooltip
* Exp Cost when repair
* Enchantability of tool
* Configable
